وانەی یەکەم: Hello World (Console App) ▼
یەکەمین بەرنامە لە زمانەکەدا بۆ ناسین و پیشاندانی دەقی Hello World لەسەر شاشەی ڕەش.
#include <iostream>
using namespace std;
int main()
{
    cout << "Hello World!\n";
}
وانەی دووەم: تەمەن و ساڵی لەدایکبوون (Variables & Data Types) ▼
ڕوونکردنەوەی جۆرەکانی گۆڕاوەکان وەک (int, float, double, char, string, bool).

using namespace std;

{

وانەی سێیەم: حاسیبە ١ (سادە) ▼
کرداری کۆکردنەوە و دابەشکردنی سادە لە نێوان ژمارەکاندا.
#include <iostream>
using namespace std;
int main()
{
    int num1 = 5;
    int num2 = 4;
    int sum = num1 + num2;

    double num3 = 7;
    int num4 = 2;
    double Div = num3 / num4;

    cout << "sum:" << sum << endl;
    cout << "Div:" << Div << endl;

    system("pause");
    return 0;  
}
وانەی چوارەم: حاسیبە ٢ (کردارە بیرکارییەکان) ▼
پیشاندانی کردارەکانی کۆکردنەوە، لێدەرکردن، و لێکدان لەسەر گۆڕاوەکان.
#include <iostream>
using namespace std;
int main()
{
    int a = 8, b = 3;
    cout << "a+b" << (a + b) << endl;
    cout << "a-b" << (a - b) << endl;
    cout << "a*b" << (a * b) << endl;
    cout << "a+b" << (a + b) << endl;

    system("pause");
    return 0;  
}
وانەی پێنجەم: مامەڵەکردن لەگەڵ String ▼
وەرگرتنی پیتی دەستنیشانکراو لە تێکستێکدا و پیشاندانی درێژی تێکستەکە.
#include <iostream>

int main()
{

وانەی شەشەم: دروستکردنی فانکشن (Functions) ▼
چۆنیەتی دروستکردنی فانکشنی دەرەکی و ناردنی زانیاری (Arguments) بۆی.
#include <iostream>

void sayHi(string name, int age);

int main()
{

{

}
وانەی حەوتەم: وەرگرتنی داتا لە بەکارهێنەر (cin Input) ▼
بەکارھێنانی فەرمانی cin بۆ وەرگرتنی زانیاری ناو و تەمەن لە ڕێگەی کیبۆردەوە.

وانەی هەشتەم: مەرجەکان (If, Else if, Else) ▼
بەراوردکردنی دوو ژمارە کە ئایا کامیان گەورەترن، بچووکترن یان یەکسانن.
#include <iostream>
using namespace std;

int main()
{
    int num1;
    int num2;
    cin >> num1;
    cin >> num2;
    if(num1 > num2)
    {
        cout << "zhmaray yakam gawra tra la dwam" << endl;
    }
    else if (num1 < num2) {
        cout << "zhmaray yakam bchwktra tra la dwam" << endl;
    }
    else if (num1 == num2) {
        cout << "zhmaray yakam yaksana ba dwam" << endl;
    }
    else {
        cout << "hichkamyan" << endl;
    }
    system("pause");
    return 0;  
}
وانەی نۆیەم: خولگەکان (For Loop) ▼
بەکارھێنانی For Loop بۆ دووبارەکردنەوەی کۆد لە ١ تا ٤.
#include <iostream>
using namespace std;

int main()
{
    for (int x = 1; x < 5; x++) {
        cout << "slaw " << x << endl;
    }
    system("pause");
    return 0;  
}

وانەی یازدەیەم: دروستکردنی حاسیبە بە MFC Dialogue Basic ▼
ڕێنمایی دروستکردنی پڕۆژەی حاسیبە:
1. دروستکردنی ٣ Static Text بە ناوەکانی (A, B, C).
2. دروستکردنی ٣ Edit Control و گۆڕینی IDـەکانیان بۆ (_A, _B, _C).
3. زیادکردنی Variable لە جۆری double بۆ هەریەکێکیان لەسەر کاتی Value بە ناوەکانی m_a و m_b و m_c.
4. دروستکردنی Event Handler بۆ دوگمەکە و نووسینی ئەم کۆدە: 
UpdateData(TRUE);

m_c = m_a + m_b;

UpdateData(FALSE);
